Gerador de Prompt

Esta aplicação permite gerar um prompt com base nos depoimentos de NPS inseridos. Basta preencher o campo abaixo com os depoimentos e clicar no botão "Gerar Prompt". O resultado será exibido na caixa de texto abaixo, e aí é só clicar em "Copiar Prompt" e jogar lá no Chat GPT. A ideia deste gerador é realizar análises simples de NPS. Esta aplicação não possui backend e não salva nenhuma informação, porém, recomendo que remova informações confidenciais dos depoimentos antes de enviar ao ChatGPT. Para que o gerador quebre a linha corretamente do depoimento, é necessário que inclua "##" ao fim do comentário
